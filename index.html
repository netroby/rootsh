You need wget or curl installed, choose as your like.

Update ssh for your linux

wget -qO-   s.netroby.com/update-ssh.sh   | sh

Update Git for your linux

wget -qO- s.netroby.com/update-git.sh   | sh

Update Emacs for your linux

wget -qO- s.netroby.com/update-emacs.sh   | sh

Generate tls self signed cert for nginx

wget -qO- s.netroby.com/gentlscert.sh  | sh -s "*.netroby.com"

Install hack fonts for linux

wget -qO- s.netroby.com/hackfonts.sh  | sh
